# Korean ASO Metadata — ParentKin / 다독

## App Name (30 char max)
다독 - AI 육아 코치

## Short Description (80 char max)
과학 기반 AI 육아 코치. 24시간 언제든 실용적인 육아 조언.

## Full Description (4000 char max)

다독은 스트레스받은 부모를 위한 AI 육아 코치입니다. 
실제 육아 전문가들의 책과 연구를 바탕으로, 지금 당장 필요한 조언을 제공합니다.

### 🎯 다독만의 특별함

**📚 15권 이상의 전문 육아 도서 기반**
- 『전뇌 아이』(The Whole-Brain Child) - 다니엘 시겔 & 틴아 브라이슨
- 『노드라마 디시플린』(No-Drama Discipline) - 시겔 & 브라이슨
- 『아이들이 들을 때까지 어떻게 말할 것인가』(How to Talk So Kids Will Listen) - 아델 파버 & 일레인 마즐리쉬
- 『1-2-3 매직』(1-2-3 Magic) - 토마스 펠란
- 『폭발하는 아이』(The Explosive Child) - 로스 그린
- 외 280개 이상의 전문 자료

**🎤 음성 또는 글로 질문하세요**
아이가 울음을 터뜨린 순간, 말로 설명하기 어려울 때 녹음 버튼 하나로 충분합니다.

**⚡ 즉시 실용적인 조언**
✅ 이렇게 해보세요 - 구체적인 행동과 대화 스크립트
❌ 이렇게는 하지 마세요 - 피해야 할 반응
🧠 왜 이렇게 해야 하나요 - 아이 발달 심리에 기반한 설명
📖 출처 - 어떤 책의 어떤 방법인지 명확하게

**🔍 더 깊이 (Go Deeper)**
기본 조언 후 더 깊이 이해하고 싶다면, 750단어 분량의 개인화 리포트를 제공합니다.

### 💬 다독이 기억합니다
연속된 질문에 맥락을 유지합니다. "밤에 자다 깨면요?" 같은 추가 질문에도 앞서 물은 상황을 기억하고 답합니다.

### 🌟 부모들의 이야기
"3살 아이가 마트에서 울음을 터뜨렸을 때, 다독이 알려준 대로 하니 5분 만에 진정됐어요" 
- 2살, 5살 아이맘

### 🔒 안전 우선
- 의료 관련 질문은 소아과 의사 상담을 권합니다
- 아동 학대나 위험 상황 즉시 감지하고 지원 정보를 제공합니다
- 체벌은 어떤 상황에서도 권장하지 않습니다

### 📱 언제 어디서나
- 24시간 언제든 사용 가능
- 한국어/영어 지원
- 다크 모드 지원
- 즐겨찾기로 유용한 조언 저장
